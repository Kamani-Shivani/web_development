CSS Selectors

Applying CSS to Different Parts of HTML

This is a paragraph targeted by tag name.

  1. The element selector targets elements based on theit HTML tag name

  2. Class selector target elements based on the value of the class attribute.

  3. ID selectors target elements based on the value of the id attribute

  4. Attribute selector target elements based on the their attributes and values.

  5. The universal selector targets all elements.
